# Car Finance Reclaim

---

Car Finance Reclaim.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v22.12.0
```

```bash
npm -v
# 11.0.0
```

## Installation

Run the commands:

```bash
npm install
```

```bash
npm run dev
```
